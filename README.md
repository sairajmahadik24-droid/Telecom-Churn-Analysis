Telecom Customer Churn Analysis

Project Overview : 
Customer churn is one of the biggest problems faced by telecom companies. 
This project analyzes telecom customer data to identify key factors responsible for customer churn and helps businesses improve retention strategies.

The project performs data cleaning, exploratory data analysis (EDA), and visualization to extract business insights.


# Business Objectives

- Identify the main factors causing customer churn
- Analyze the relationship between customer charges and churn
- Evaluate contract types and their impact on retention
- Provide data-driven recommendations for reducing churn

---

# Dataset Information

The dataset contains telecom customer information including:

- Customer demographics
- Contract type
- Monthly charges
- Tenure
- Payment method
- Churn status


# Analysis Performed

Data Cleaning
- Handling missing values
- Converting data types
- Removing duplicates

Exploratory Data Analysis

- What percentage of customers churn?  
- Which contract type has highest churn?  
- Do high monthly charges lead to churn?  
- Which payment method causes more churn?

# Visualizations

- Churn distribution
- Contract type vs churn
- Monthly charges vs churn
- Tenure vs churn
- Payment method vs churn


# Key Insights

- Customers with **month-to-month contracts churn the most**
- Higher **monthly charges increase churn probability**
- Customers with **longer tenure have lower churn**
- **Electronic check payment users churn more**
